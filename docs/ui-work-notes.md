# Front UI 작업 정리

작성일: 2026-06-06

## 현재 방향

메인화면은 졸업작품 기능 설명서처럼 보이기보다, 일반 SaaS 홍보 페이지처럼 깔끔하게 유지한다.

로그인 전 화면은 서비스 소개와 로그인 진입 중심으로 구성하고, 실제 백엔드 기능은 로그인 이후 관리자 화면에서 자연스럽게 드러나게 한다.

## 현재까지 반영한 변경사항

### 1. 메인화면

- 스크린샷 폴더의 `메인화면2` 이미지 느낌을 참고해 메인 랜딩 페이지를 재구성했다.
- 기존의 AI가 만든 듯한 단순한 카드형 느낌을 줄이고, 중앙 정렬형 SaaS 랜딩 페이지 느낌으로 변경했다.
- 색상은 검은색 + 흰색 중심에서 회색 + 아주 약한 하늘색 톤으로 조정했다.
- 주차 현황 미리보기의 차량 표시를 단순 막대가 아니라 차량처럼 보이도록 CSS 형태를 개선했다.
- 푸터는 관리자 업무 흐름이 보이도록 정리했다.

관련 파일:

- `src/pages/public/MainPage.jsx`
- `src/components/common/PublicFooter.jsx`
- `src/styles/global.css`

### 2. 관리자 공통 UI 1단계

로그인 이후 전체 관리자 화면의 공통 틀만 먼저 메인화면 톤에 맞춰 수정했다.

- 사이드바를 어두운 남색 배경에서 밝은 관리자 콘솔 스타일로 변경했다.
- 브랜드명을 `Park On`으로 정리했다.
- 상단바에 관리자 역할과 사용자 이름 영역을 추가했다.
- 카드, 테이블, 버튼, 입력창의 기본 스타일을 회색 + 약한 하늘색 톤으로 통일했다.
- 모바일 화면에서 사이드바와 상단바가 어색하게 고정되지 않도록 반응형 기본값을 보정했다.

관련 파일:

- `src/components/layout/Header.jsx`
- `src/components/layout/Sidebar.jsx`
- `src/styles/global.css`

### 3. 백엔드 기능과 연결되는 프론트 보완

백엔드 `graduation`의 web 관련 기능 중 프론트에서 바로 연결되는 부분을 기준으로 일부 누락 화면을 보완했다.

- 웹 관리자 화면에서 아파트 관리자 상세 정보를 볼 수 있도록 정리했다.
- 아파트 관리자 마이페이지에서 관리자 이름, 이메일, 전화번호를 수정할 수 있도록 보완했다.
- 입주민 수정, 차량 등록/수정 쪽에서 상세 조회 API를 직접 활용하도록 정리했다.

관련 파일:

- `src/api/apartmentManagerApi.js`
- `src/api/webAdminApi.js`
- `src/contexts/ApartmentManagerContext.jsx`
- `src/pages/apartmentManager/MyPage.jsx`
- `src/pages/apartmentManager/ResidentEdit.jsx`
- `src/pages/apartmentManager/VehicleForm.jsx`
- `src/pages/webAdmin/ApartmentManagerManagement.jsx`

## 앞으로 변경할 UI 지침

### 전체 원칙

- 메인화면은 일반 SaaS 홍보 페이지처럼 유지한다.
- 로그인 이후 관리자 화면은 실제 백엔드 기능을 기준으로 정리한다.
- 색상은 회색 + 아주 약한 하늘색 + 검은색 포인트를 기본으로 유지한다.
- 너무 화려한 그라데이션이나 장식 요소는 피한다.
- 카드 안에 또 카드를 넣는 구조는 줄이고, 섹션과 테이블 중심으로 깔끔하게 정리한다.
- 발표 시 설명하기 쉬운 화면 구조를 우선한다.

### 레이아웃 지침

- 관리자 화면은 좌측 사이드바 + 상단바 + 본문 구조를 유지한다.
- 대시보드는 핵심 지표 카드, 최근 목록, 처리 필요 항목 순서로 배치한다.
- 목록 화면은 검색/필터/테이블/페이지네이션 흐름을 일관되게 유지한다.
- 등록/수정 화면은 입력 그룹을 명확히 나누고, 하단에 저장/취소 버튼을 고정된 흐름으로 배치한다.
- 상세 화면은 `기본 정보`, `상태 정보`, `관리 작업`처럼 발표하기 쉬운 구역으로 나눈다.

### 기능별 방향

웹 관리자:

- 대시보드: 가입 승인 대기, 전체 관리자 수, 문의 현황 중심으로 정리
- 가입 승인: 승인/거절 작업이 가장 잘 보이도록 테이블과 상세 영역 정리
- 관리자 정보 관리: 상세 보기와 관리 버튼을 한곳에 모아 유지
- 문의 관리: 답변 상태와 처리 흐름을 명확히 표시

아파트 관리자:

- 대시보드: 입주민, 차량, 주차구역, 알림, 문의 상태를 한눈에 보이게 정리
- 마이페이지: 아파트 정보와 관리자 정보를 분리해서 표시
- 입주민 관리: 동/호수, 이름, 연락처, 차량 연결 상태 중심
- 차량 관리: 차량번호, 소유자, 상태, 등록일 중심
- 주차 관리: 주차구역, 주차면, 입출차/차단기 상태가 명확히 보이게 정리
- 알림/문의: 읽음/미읽음, 답변 대기/완료 상태가 바로 보이도록 정리
